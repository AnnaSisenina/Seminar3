Для того, чтобы выполнить коммит, используется команда *git commit*. Для этого в терминале с папкой-репозиторием написать *git commit - m "<сообщение к коммиту>"*. Сообщение к коммиту писать ***ОБЯЗАТЕЛЬНО***!!!
